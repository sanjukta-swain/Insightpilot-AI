# 📊 InsightPilot AI

An AI-powered Business Analytics and Decision Intelligence Platform built using Python, Streamlit, Machine Learning, and Google Gemini AI.

## 🚀 Features

- 📂 Upload CSV/Excel datasets
- 🧹 Automatic Data Cleaning
- 📊 Interactive Business Dashboard
- 🤖 AI-Generated Business Insights
- 📈 Sales Forecasting (Next 6 Months)
- 📄 PDF Report Generation
- ⬇️ Download Cleaned Dataset
- 🎛️ Dashboard Filters

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- Google Gemini AI
- ReportLab


## 📌 Project Highlights

- AI-powered business insight generation using Google Gemini AI
- Automatic data cleaning and preprocessing
- Interactive dashboard with filters and visualizations
- Machine Learning-based Sales Forecasting
- Export AI-generated business reports as PDF
- Download cleaned dataset in CSV format
